# **Museum Landing Page**

A responsive landing page for an art museum, implemented according to a Figma design.
The project focuses on pixel-perfect layout, semantic HTML, SCSS architecture, and clean UI interactions without using frontend frameworks.

---

## **LIVE DEMO**

**View the live website:**
👉 https://iirk1.github.io/Museum/

---

## **Technologies**

- **HTML5** — semantic, accessible markup
- **SCSS (Sass)** — modular styles using BEM methodology
- **Vanilla JavaScript** — UI interactions and basic logic
- **BEM** — structured and maintainable CSS architecture
- **Google Fonts** — IBM Plex Sans, Montserrat
- **Git & GitHub Pages** — version control and deployment

---

## **Installation**

### **Clone the repository**

```bash
git clone https://github.com/your-username/Museum.git
cd Museum
```

### **Install dependencies**

```bash
npm install
```

### **Run the project locally**

```bash
npm start
```

## **Features**

- **Fully responsive layout** (desktop / tablet / mobile)
- **Pixel-perfect implementation** based on Figma design
- **Burger menu** for mobile navigation
- **Smooth scrolling** across the entire page
- **Gallery section with slider** (mobile-friendly)
- **Hover animations** for gallery and exhibition images
- **Interactive subscription form** with validation and auto reset
- **Clickable navigation links** leading to corresponding sections
- **Social media links** opening in a new tab
- **Clean, maintainable codebase** using BEM and SCSS
- **Performance- and UX-oriented implementation**
